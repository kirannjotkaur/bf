What is TypeScript?

TypeScript is a superset of JavaScript that adds static types to the language. It allows developers to define variables, functions, and objects with a specific type, helping catch errors early during development.

TypeScript compiles to JavaScript and can be used alongside existing JavaScript codebases. Its main feature is type checking, which helps avoid runtime errors and improves code maintainability and readability.

Installing TypeScript

Follow these steps to install TypeScript:

  • Ensure you have Node.js installed. Check by running node -v in your terminal.
  • Install TypeScript globally using npm:
    • npm install -g typescript
  • Check if TypeScript was installed successfully by running:
    • tsc -v

    This should display the version of TypeScript installed.

let message: string = "Hello, TypeScript!";

This code defines a variable message with a type of string and prints it to the console.

To compile the TypeScript file into JavaScript, run:

tsc hello.ts

This will generate a hello.js file, which you can run with:

node hello.js

You should see Hello, TypeScript! printed in the console.
