What is Express.js?

Express.js is a fast, unopinionated, and minimalist web framework for Node.js. It simplifies routing, handling requests, and building web applications.

Express is widely used to build web applications and APIs due to its flexibility and large ecosystem of plugins and middleware.

It helps you quickly set up server-side routing, middleware, and handle HTTP requests with ease.

Installing Express.js

Follow these steps to install Express.js:

  • Make sure you have Node.js installed. You can verify this by running node -v in your terminal.
  • Create a new Node.js project folder:
    • mkdir my-express-app
    cd my-express-app
  • Initialize a new Node.js project:
    • npm init -y
  • Install Express.js via npm:
    • npm install express --save

Once the installation is complete, Express will be available for use in your project.

Basic Express.js Example

Let's create a simple Express.js server. Create a new file named app.js and add the following code:

const express = require('express');
const app = express();

// Define a route for the homepage
app.get('/', (req, res) => {
    res.send('Hello, Express.js!');
});

// Start the server on port 3000
app.listen(3000, () => {
    console.log('Server running at http://localhost:3000/');
});

Run the server by typing the following command in your terminal:

node app.js

Now, open your browser and go to http://localhost:3000/, and you should see the message "Hello, Express.js!"
