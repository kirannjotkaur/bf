What is Node.js?

Node.js is a runtime environment that allows you to run JavaScript code on the server-side, enabling you to build fast, scalable network applications.

Node.js uses an event-driven, non-blocking I/O model, making it lightweight and efficient. It is ideal for building real-time applications like chat applications, API services, and more.

Installing Node.js

To get started with Node.js, you need to install it on your machine. Follow these steps:

  • Go to the official Node.js website.
  • Download the latest LTS (Long Term Support) version for your platform (Windows, macOS, or Linux).
  • Run the installer and follow the instructions to complete the installation.

After installation, verify Node.js is correctly installed by running the following commands in your terminal:

node -v

This will display the installed version of Node.js. You can also verify npm (Node Package Manager) by running:

npm -v

Basic Node.js Example

Let's create a simple HTTP server using Node.js. Create a new file named server.js and add the following code:

const http = require('http');

// Create an HTTP server and listen on port 3000
http.createServer((req, res) => {
    res.writeHead(200, {'Content-Type': 'text/plain'});
    res.end('Hello, Node.js!');
}).listen(3000, () => {
    console.log('Server running at http://localhost:3000/');
});

Save the file and run the server using the following command:

node server.js

Visit http://localhost:3000/ in your browser, and you should see the message "Hello, Node.js!"
